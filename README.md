# 🌊 Environmental Stress Movement Score (ESMS)

This project explores how environmental stressors, specifically ocean temperature anomalies, influence movement behavior in marine species.

## 🚀 Key Idea
Movement patterns may act as early indicators of environmental stress.

## 📊 Methods
- NOAA Sea Surface Temperature Data
- Marine animal tracking data (Movebank)
- Feature engineering (speed variability, path deviation)
- Modeling (Linear Regression, Random Forest, Clustering)

## 🛠 Tech Stack
Python (pandas, NumPy, scikit-learn, matplotlib)
